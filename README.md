Elite Flow Plumbing is a high-performance, conversion-optimized landing page designed for premium plumbing services. This project implements FAANG-level design standards, featuring sophisticated animations with GSAP and Framer Motion, and a robust component architecture.

## ✨ Key Features

- **🏆 Premium UI/UX**: Dark-themed luxury aesthetic with gold accents and high-end typography.
- **⚡ Performance-First**: Built on Vite with React 19 for blazing-fast load times and smooth interactions.
- **📱 Fully Responsive**: Seamless experience across all devices, from ultra-wide monitors to mobile handhelds.
- **🎨 Advanced Animations**: Integrated GSAP and Framer Motion for subtle, buttery-smooth micro-interactions.
- **🚀 Conversion Optimized**: Strategic CTA placement and persuasive copywriting designed to drive leads.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Bundler**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) & [Vanilla CSS](https://developer.mozilla.org/en-US/docs/Web/CSS)
- **UI Components**: [Radix UI](https://www.radix-ui.com/)
- **Animation**: [GSAP](https://gsap.com/) & [Framer Motion](https://www.framer.com/motion/)
- **Form Handling**: [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18.0 or higher)
- npm or yarn

### Installation

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/Ismail-Khan-Dev/elite-flow-plumbing-website.git
    cd elite-flow-plumbing-website
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Start Development Server**:
    ```bash
    npm run dev
    ```

4.  **Build for Production**:
    ```bash
    npm run build
    ```
